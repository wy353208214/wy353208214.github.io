Sublime快捷键

发表于 2018-01-19   |     |   阅读次数

Sublime快捷键

插入
在下一行插入新行。即使光标不在行尾，也能快速向下插入一行

Ctrl+Enter

在上一行插入新行。即使光标不在行首，也能快速向上插入一行

Ctrl+Shift+Enter

复制光标所在整行，插入到下一行。即先将光标所在行复制，然后再插入到光标所在的下一行

Ctrl+Shift+D

编辑
合并选中的多行代码为一行。例如：将多行格式的CSS属性合并为一行。

Ctrl+J

粘贴并保持缩进

Ctrl+Shift+V

使用标签包裹一行，Web开发利器

Alt+Shift+W

移除与光标所在位置相关的父标签。对清除标记很有帮助。

Ctrl+Shift+;

按Ctrl，依次点击或选取，可需要编辑的多个位置

Ctrl + 左键点击

Esc 退出光标多行选择，退出搜索框，命令框等

Ctrl+T 左右字母互换

开启/关闭侧边栏

Ctrl+K+B

选择
选中光标所占的文本，继续操作则会选中下一个相同的文本

Ctrl+D

选中文本按下快捷键，即可一次性选择全部的相同文本进行同时编辑。例如：快速选中并更改所有相同的变量名、函数名等。

Alt+F3

选中整行，继续操作则继续选择下一行，效果和 Shift+↓ 效果一样。

Ctrl+L

先选中多行，再按下快捷键，会在每行行尾插入光标，即可同时编辑这些行。

Ctrl+Shift+L

选择括号内的内容（继续选择父括号）。例如：快速选中删除函数中的代码，重写函数体代码或重写括号内里的内容。

Ctrl+Shift+M

向上选中多行。

shift+↑

向下选中多行。

shift+↓

向左选中文本。

Shift+←

向右选中文本。

Shift+→

向左单位性地选中文本。

Ctrl+Shift+←

向右单位性地选中文本。

Ctrl+Shift+→

移动
光标移动至括号内结束或开始的位置。

Ctrl+M

向左单位性地移动光标，快速移动光标。

Ctrl+←

向右单位性地移动光标，快速移动光标。

Ctrl+→

输入数字跳转到指定行。与 Ctrl + P 输人 ： 再输入数字一样的功能。

Ctrl+G

折叠
选中代码，按下快捷键，折叠代码。

Ctrl+Shift+[

选中代码，按下快捷键，展开代码。

Ctrl+Shift+]

展开所有折叠代码。

Ctrl+K+0

缩进
向右缩进。

Tab

向左缩进。

Shift+Tab

向右缩进。

Ctrl+]

向左不缩进。

Ctrl+[

注释
注释单行。

Ctrl+/

注释多行。

Ctrl+Shift+/

删除
从光标处开始删除代码至行尾。

Ctrl+K+K

删除整行。

Ctrl+Shift+K

撤消
撤销。

Ctrl+Z

恢复撤销。

Ctrl+Y

软撤销，撤销光标位置。

Ctrl+U

大小写转换
转换大写。

Ctrl+K+U

转换小写。

Ctrl+K+L

查找
打开底部搜索框，查找关键字。如果先选中文本再按快捷键，则自动开始查找。

Ctrl+F

查找并替换。

Ctrl+H

在文件夹内查找，允许添加多个文件夹进行查找，同时支持替换。

Ctrl+shift+F

打开搜索框。该搜索框有强大的搜索功能：

Ctrl+P

打开搜索框，自动带@，输入关键字，查找文件中的函数名。举个栗子：在函数较多的页面快速查找某个函数。

Ctrl+R

1、输入当前项目中的文件名，快速搜索文件，
2、输入@和关键字，查找文件中函数名，
3、输入：和数字，跳转到文件中该行代码，
4、输入#和关键字，查找变量名。

打开搜索框，自动带#，输入关键字，查找文件中的变量名、属性名等。

Ctrl+：

打开命令框。场景栗子：打开命名框，输入关键字，调用sublime text或插件的功能，例如使用package安装插件。

Ctrl+Shift+P

分屏
窗口分屏，恢复默认1屏（非小键盘的数字）。

Alt+Shift+1

左右分屏-2列。

Alt+Shift+2

左右分屏-3列。

Alt+Shift+3

左右分屏-4列。

Alt+Shift+4

等分4屏。

Alt+Shift+5

垂直分屏-2屏。

Alt+Shift+8

垂直分屏-3屏。

Alt+Shift+9
