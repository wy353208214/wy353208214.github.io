linux常用功能

  1. 安装卸载软件，比如mysql
    • 首先更新下仓库：yum install -y update
    • 安装软件：yum install mysql-server mysql-client
    • 卸载软件：
      1. yum remove mysql-server
      2. yum remove mysql-client
      3. yum autoremove彻底删除，清理残留

  2. Linux 常用命令
    md5校验：md5Sum xxx.txt
    全选删除:%d、按ggdG
    定位：

    • 快速到最后一行：shift + G
    • 快速到第一行： 1 + shift +G
    • 快速到第n行： n + shift + G
